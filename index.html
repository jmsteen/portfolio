My work

Projects

Whether it's a thoughtful UI, a performant back end or a delightful feature, each of my projects uses a variety of technologies to enable a seamless and useful user experience.

Austana

A project management web application inspired by Asana where users can create and organize projects, tasks and teams.

  • Utilized Redux and asynchronous JavaScript with a RESTful API to provide a seamless user experience across all React components.
  • Incorporated Webpack and Babel to ensure UI cross-browser compatibility.
  • Implemented backend and frontend authentication with Rails models, BCrypt and React Router to enhance user privacy.

Roastia

Roastia allows indie coffee enthusiasts to find craft coffee shops closest to them.

  • Integrated Mongoose, MongoDB, GraphQL queries and React Autosuggest to design a partial text search, which allows users to search for coffee shops and coffees based on a variety of attributes and view updated suggestions as they type.
  • Used React Hooks and query strings to create a shareable URL with relevant search parameters upon completion of a search.

Super

About Me

I'm a software engineer focused on creating empathic products.

I've always been most interested in the fusion of the analytical and the creative. After college, I worked as a CPA at accounting firms KPMG and PwC, while I'd spend my weekends working on creative projects like music production and podcasting.

Once I discovered software engineering, I never looked back. It was thrilling to see projects come to life across the full stack using technologies like Ruby, Rails, JavaScript, React and Redux.

I also utilized MongoDB, Express, Node, Apollo GraphQL, PostgreSQL, Mongoose, Docker, ES6 and SCSS to create the projects above. I'm currently learning Python through practice of data structures and algorithms.

Outside of programming, I love listening to podcasts, making music, playing with my German Shepherd and geeking out on health and nutrition.
